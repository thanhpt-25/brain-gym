- [ ] **Database query optimization** — slow-query log clean

### Data Consistency & Idempotency

- [ ] **Canary auto-pause is idempotent**
  - Re-trigger same rollback-rate scenario: no duplicate flag records
  - Variant state unchanged (apply counts same)
- [ ] **Reputation flag creation is idempotent**

- [ ] **KG overlap recompute debounce works**
  - Rapid question edits (3 in 10s): single job enqueued
  - Verified via queue inspection + compute-time logs
