# Brain Gym (CertGym) — Product Overview

## Purpose
Brain Gym (branded as "CertGym") is a community-powered certification exam training platform — a "gym for the brain." Users practice with mock exams for international IT certifications, contributors create questions for the community, and the system acts as a data-driven coach tracking progress, identifying weaknesses, and suggesting study strategies.

## Value Proposition
- Community-driven question bank with voting, review, and expert verification
- Realistic exam simulation (timer, navigation, mark-for-review) matching real exam formats
- Smart analytics with domain breakdown, weak topic detection, and pass probability
- Multiple learning modes: exam simulation, study mode, flashcards

## Key Features
- **Question System**: MCQ questions with single/multiple correct answers, explanations, reference links, difficulty levels, tags, scenario-based questions, code snippets, and image support
- **Exam Builder**: Create mock exams with configurable question count, time limits, difficulty distribution, and visibility (public/private/link-shared)
- **Exam Simulation Engine**: Countdown timer, question navigation, mark-for-review, submit and review answers with score, domain breakdown
- **Result Analytics**: Exam history, score trends, pass probability, weak topics, most-missed questions
- **Community Knowledge**: Comment threads on questions, explanation improvements, voting (upvote/downvote), reporting (wrong answer, outdated, duplicate)
- **Quality Control**: Expert verification badges, report system, voting system
- **Gamification**: Points for creating/reviewing questions, badges (Exam Creator, Cloud Master, Top Contributor), leaderboards
- **Authentication**: JWT-based auth with access/refresh token flow, role-based access (Learner, Contributor, Reviewer, Admin)
- **Adaptive Exam** (planned): Difficulty adjusts based on answer correctness

## Target Certifications
- AWS Certified Solutions Architect
- Microsoft Azure Fundamentals
- Google Professional Cloud Architect
- PMP Certification
- Certified Kubernetes Administrator

## Target Users
1. **Learners** — Take mock tests, view explanations, track progress
2. **Contributors** — Create and share questions/exams
3. **Reviewers/Experts** — Verify questions, improve explanations, flag errors
4. **Admins** — Manage certifications, users, moderation
